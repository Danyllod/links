# links

Pagina de links pessoais estilo Linktree — feita com HTML e CSS puro.

## Estrutura

```
links/
├── index.html   # Pagina principal
├── style.css    # Estilos (tema escuro, acento roxo)
└── README.md
```

## Funcionalidades

- Avatar automatico via GitHub
- Botoes de link com hover animado
- Tema escuro com acento roxo
- Layout responsivo (mobile-first)
- Fonte Inter (Google Fonts)

## Como usar

1. Edite os links em `index.html`
2. Hospede no GitHub Pages, Vercel ou Netlify
